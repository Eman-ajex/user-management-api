# user-management-api
Spring Boot REST API with JWT Authentication, MySQL, and CRUD operations for user management  
# Features
- User Registration
- User Login
- JWT Token Generation
- Password Encryption (BCrypt)
- CRUD Operations
## Technologies
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- Maven
- ## API Endpoints

- POST /auth/register

- POST /auth/login

- GET /users

- GET /users/{id}

- PUT /users/{id}

- DELETE /users/{id}
